# HOMEPAGE
## As soon as you load the website your location access will be asked as we have designed this website for users to access hospitals located in Goa.
![Screenshot (110)](https://user-images.githubusercontent.com/122557922/232237149-78b70026-cca6-443e-b62c-3890599c1d7d.png)

## You can create your account by clicking on the account icon on the navigation bar
![Screenshot (102)](https://user-images.githubusercontent.com/122557922/232237147-2159322a-cb1d-4661-9421-27e3f792a3ff.png)

## You can upload your reports or previous medical history by clicking on the upload button in the navigation bar for easy access.
![Screenshot (125)](https://user-images.githubusercontent.com/122557922/232240074-d633ff71-ff5f-4f25-a3fe-01309bed8ba0.png)

## Then the user can navigate to various sections of the page from the linked dropdown menu available on the navigation bar.
## If a user wants to consult a doctor then he can click on the doctor's option from the dropdown menu which will take him to a section which has all the info about doctors.
